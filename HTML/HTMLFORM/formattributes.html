3. Quick Attribute Table

Attribute Used On Purpose
action <form> Sets where form data is sent.
method <form> Sets how data is sent, usually get or post.
name Form controls Sets the key used when form data is submitted.
id Any element Connects labels, CSS, and JavaScript to an element.
required Inputs, selects, textareas Makes a field mandatory.
placeholder Inputs, textareas Shows hint text inside a field.
min, max, step Number, range, date, time inputs Controls allowed value limits.
pattern Text-like inputs Checks the value using a regular expression.
disabled Form controls Disables a field and usually prevents submission.
readonly Text-like inputs, textarea Makes a field non-editable but still submitted.
