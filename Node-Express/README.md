# Node.js and Express.js Learning

Node.js and Express.js are used to create the backend server in the MERN stack.

## What To Learn

- What Node.js is
- npm and package management
- `package.json`
- CommonJS and ES modules
- Creating a basic server
- Express app setup
- Routes
- Controllers
- Middleware
- Request and response objects
- Route parameters
- Query parameters
- JSON body parsing
- Error handling
- Environment variables
- REST API structure
- MVC folder pattern
- Connecting backend with frontend

## Practice Projects

- Basic Express server
- Notes API
- Todo API
- Recipe API
- Blog API
- User API

## Goal

I should be able to create clean backend APIs that can send, receive, validate, and process data.
