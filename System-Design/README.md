# System Design

This folder is for learning how large applications are designed.

## What To Learn

- Client-server architecture
- Horizontal and vertical scaling
- Load balancers
- Caching
- Database replication and sharding
- Queues and async processing
- Rate limiting
- File storage
- Search basics
- Observability
- Reliability and fault tolerance

## Practice

- Design a URL shortener.
- Design a chat app.
- Design an e-commerce system.
- Design a notification system.
- Design a file upload system.

## Goal

I should be able to explain practical system design choices for real applications.
