# Spring Boot Backend

This folder is for Java backend development using Spring Boot.

## What To Learn

- Spring Initializr project setup
- Maven or Gradle
- Controllers and REST endpoints
- Services and business logic
- Repositories and Spring Data JPA
- DTOs and request validation
- Exception handling
- Environment configuration
- Profiles for dev, test, and production
- Logging
- API documentation with Swagger/OpenAPI

## Practice Projects

- Basic REST API
- Todo API
- Notes API
- Blog API
- Product API
- Authentication API

## Goal

I should be able to build clean Spring Boot APIs that connect to a database, validate data, handle errors, and support real frontend applications.
