# JavaScript Learning

JavaScript is the programming language of the MERN stack. Learn it deeply before jumping into React.

## What To Learn

- Variables: `let`, `const`, and scope
- Data types: string, number, boolean, null, undefined, object, array
- Operators and expressions
- Conditional statements
- Loops
- Functions and arrow functions
- Arrays and array methods
- Objects
- DOM selection and manipulation
- Events
- Forms and validation
- JSON
- Fetch API
- Promises
- `async` and `await`
- Error handling
- ES6+ features
- Modules: import and export

## Practice Projects

- Counter app
- Color changer
- Calculator
- Todo app
- Form validator
- Weather app using API
- Quiz app

## Goal

I should be able to build interactive frontend features and understand the JavaScript logic used inside React and Node.js.
