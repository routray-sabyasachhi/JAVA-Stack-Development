# Databases

This folder is for database learning for Java full-stack development.

## Main Focus

- SQL fundamentals
- PostgreSQL or MySQL
- Tables and relationships
- Primary keys and foreign keys
- Joins
- Indexes
- Transactions
- Normalization basics
- Query optimization basics
- Spring Data JPA mapping
- Database migrations with Flyway or Liquibase

## NoSQL Awareness

MongoDB can still be learned later when a project needs document storage, but the main Java full-stack path should start with SQL databases.

## Practice Projects

- Student management schema
- Notes app database
- Blog database
- E-commerce product and order schema
- User and role schema

## Goal

I should be able to design reliable relational databases and connect them to Spring Boot applications.
